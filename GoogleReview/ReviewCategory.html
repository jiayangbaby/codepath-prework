Google Review Analysis

Project Overview

Normalized and vectorized raw data during a text mining project for Google Review, calculated perplexity scores for different Latent Dirichlet Allocation models to select the best one for each star-rating. Categorized each Google Review within the same star-rating into different topics in the best model. Gave each topic meaning by summarizing unique and relevant words, so readers can understand the main complaints for one-star reviews or common compliments for five-star reviews. Google Reviews are from 6 Google Stores. 1. Hanely 2. Tower Grove 3. Woodland 4.Willow St 5. Maple St 6. University Dr
